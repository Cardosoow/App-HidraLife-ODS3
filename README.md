# HidraLife - Monitor de Hidratação Diária

**Estudante:** Iago Amorim Cardoso
**Disciplina:** Programação para Dispositivos Móveis  
**Turma:** B-0925 - GV2  

## 1. Descrição do Problema
Muitas pessoas, especialmente idosos e trabalhadores com rotina intensa, esquecem de consumir a quantidade mínima de água diária necessária. A desidratação pode levar a problemas renais, fadiga e dores de cabeça, impactando diretamente a qualidade de vida. A ausência de ferramentas simples e diretas dificulta o hábito.

## 2. Justificativa e Vinculação ODS
Este projeto foi desenvolvido para atender ao **ODS 3 (Saúde e Bem-Estar)**, especificamente contribuindo para a manutenção da saúde física através da prevenção. O aplicativo promove a conscientização sobre hábitos saudáveis de forma tecnológica e acessível.
* **Meta ODS:** Assegurar uma vida saudável e promover o bem-estar para todos, em todas as idades.

## 3. Objetivos do App
* Oferecer uma interface minimalista para registro rápido de consumo de água.
* Permitir o controle quantitativo de ingestão diária (em ml).
* Incentivar o atingimento da meta de 2 litros diários.

## 4. Público-Alvo
Jovens adultos, estudantes e trabalhadores de escritório que passam longos períodos concentrados e esquecem de se hidratar, bem como idosos que necessitam de interfaces simplificadas.

## 5. Tipo de Aplicação
**PWA (Progressive Web App) / Web App Responsivo**.  
*Justificativa:* Escolhido pela facilidade de acesso via navegador em qualquer dispositivo (Android/iOS) sem necessidade de instalação via lojas de aplicativos, democratizando o acesso.

## 6. Tecnologias Utilizadas
* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização com foco em usabilidade e contraste (Acessibilidade/UX).
* **JavaScript:** Lógica de contagem e interatividade.

## 7. Instruções de Instalação e Uso
O aplicativo foi projetado para ser executado diretamente no navegador.

**Como rodar:**
1. Baixe o arquivo `index.html` deste repositório.
2. Dê um clique duplo no arquivo para abri-lo em seu navegador padrão (Chrome, Firefox, Safari).
3. Para simular a experiência mobile no PC: Pressione `F12` e clique no ícone de celular (Device Toolbar).

**Como usar:**
1. Ao abrir o app, visualize seu contador atual (inicia em 0 ml).
2. Clique no botão azul **"+ Beber 250ml"** cada vez que consumir um copo de água.
3. Ao atingir 2000ml, você receberá uma mensagem de parabéns.
4. Utilize o botão "Reiniciar Dia" para zerar o contador na manhã seguinte.

## 8. Requisitos do Sistema
* Navegador Web atualizado (Google Chrome, Edge, Safari ou Firefox).
* Qualquer sistema operacional (Windows, Linux, macOS, Android, iOS).
